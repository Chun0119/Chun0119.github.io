Technical Components

Frontend

Unity with C# language, Decentraland with TypeScript, React with TypeScript are used. The built website product is then hosted with an AWS S3 bucket.

Backend

Node.js with MongoDB and PostgreSQL are used. The built server is then hosted with an AWS EC2 instance.

Multiplayer Connection

Photon Fusion is used to provide the multiplayer connection server/services.

Web3 Integration

thirdweb is used to provide Web3 services, such as connecting to the wallet and carrying out transactions.

Voice Chat System

Agora Voice Chat is used to provide the voice chat system service.

Avatar System

Ready Player Me is used to provide the avatar system service.
