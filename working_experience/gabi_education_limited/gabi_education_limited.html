Unity Software Developer (Freelance)

Gabi Education Limited is an organization that aims to promote educational equality through the gamification of learning. As part of my role, I was responsible for developing a series of applications designed to assist dyslexic students in learning Chinese.

  • Built 7 educational games and software including over 60 mini games, integrating ChatGPT, speech-to-text, etc., serving over 1,000 students in schools.
  • Processed the data and built a web-based platform for game data management, enabling teachers to visualize the data.

Applications Introduction

The series of applications serve different purposes, including conducting pre-screening for dyslexia and training students in Chinese language skills. The applications consist of over 60 mini-games that assess and enhance various Chinese language abilities, such as reading, writing, and speaking. All game results are recorded for further analysis.

For more detailed information about the series of applications, please visit the following link: https://gabi.hk/dyslexia-prescreeng-zh/

Technical Components

Engine

Unity with C# language is used to build mobile games.

Server Support

Cloud services are used to store users' information, such as usernames, passwords, and game results. Therefore, the application identifies users based on their accounts, and the collected game results will be further analyzed.

Web-based Data Management Platform

React is used to build a platform that facilitates the viewing and editing of data for administrators.

ChatGPT Integration

ChatGPT API is integrated to provide AI-based teaching to students.

QR Code Login

QR code is supported as one of the login methods. The applications can decode and retrieve login information from scanned QR codes.
