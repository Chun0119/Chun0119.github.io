Drift Bottle

DRIFT BOTTLE is a native Android application that tries to tackle the mental health issues of users.

  • Introduction

Introduction

In recent decade, mental health issues have become more and more serious among students. Drift Bottle is an innovative platform for users to express their emotions in a message-in-a-bottle style. It allows anonymous sharing of thoughts, such as expressing negative emotions in a story-telling way so as to relieve stress, or posting positive stories to inspire others and bring a smile to the community. Moreover, Drift Bottle provides support from expertise that could talk with and guide users.

Technical Components

Engine

Android Studio with Java language is used to build a native Android application.

Server Support

Firebase - Cloud Firestore is used to store the usersâ€™ information, such as username, password, stories shared, and messages in the conversation between the expertise and the users. Therefore, the application is identifying the users by account-based.

Content Moderation & Sentiment Analysis

Recurrent Neural Network (RNN) with TensorFlow is employed for AI with two distinct models with machine learning. It is a text classification problem to filter malicious content that contained in the stories shared by the users and to determine what mood the stories belongs to.

Service & Multimedia

Service is used for playing the background music, and some animations are added for the splash screen and loading page.

Application Details

  • Register for / Login to an account

  • Share of stories anonymously

  • Read stories of fellow peers with mood filter (choose to read happy / sad stories)

  • For pre-registered specialist
    • Provide guidance for assigned users
  • For other users
    • Carry out consultation with assigned specialist
