Introduction

Hexplore Fort is a RPG adventure game which aims to demonstrate the combination of physical and virtual gaming experience. The robot equipped is acting as the character of player in the game. The players are required to break into the mansion, fight with the guardians and capture the princess.


Website of the project: https://i.cs.hku.hk/fyp/2019/fyp19029/

Technical Components

Engine

Unity with C# language is used to build an AR Android game.

Augmented Reality (AR)

AR Core is used to provide the AR-related feature, such as plane detection and image recognition. The technology is used to augment the virtual terrain and characters in game.

AR - Plane Detection

Environmental understanding performed by ARCore can look for a horizontal plane in reality. The virtual terrain of the game is generated and augmented to the detected plane.

Programmable Robot

3D-printed Arduino-based hexapod is integrated in the project. Players can control the robot movement through the game to replace the original Vorpal gamepad controller.

AR - Image Tracking

Image recognition performed by ARCore can help the game to recognize the current position of the hexapod. This feature helps the game to achieve the simulation of reality robot walking through the virtual terrain.

Bluetooth Connection

Bluetooth Classic is equipped with the Bluetooth serial board in hexapod. It acts as a connection bridge between the game and the robot. Native Android Java code is used to handle the bluetooth command for controlling the robot movement and is converted and used as a Unity Android Plugin.

Game Details

  1. Enable bluetooth and connect hexapod

  2. Scan the environment with camera for targeting a horizontal plane to generate the map

  3. Scan the image on the body of hexapod to recognize the player character

  4. Explore the map with the robot
    • Control the robot to walk or run across the terrain
    • Pick up keys placed on the map to unlock the locked doors
  5. Fight against guardians
    • Complete quick time event (QTE) to accomplish an attack or a defense action
  6. Upgrade the power of character
    • Pick up enhancing items placed on the map
    • Make transaction with the spy by golds gained after fighting the guardians
  7. Reach where the princess is to capture the princess to win the game

Future Development

Since Hexplore Fort developed for this project is mainly for demonstrating the combination of physical and virtual gaming experience, the level system is eliminated, and Hexplore Fort is having only one fixed map. Taking reference to Tower of the Sorcerer, there are some suggestions for a long-term development.

For example:

  • Level system
  • More types of enemies
  • Skill system for enemies and player.

As times goes on, if object detection is available in ARCore, the new AR technology can be used to replace the image recognition of robot in used currently. This will greatly reduce the limitation of the game caused by ARCore.
