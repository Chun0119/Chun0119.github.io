Ray Shooter

Ray Shooter is a shooting game, which players can destroy the incoming UFO and to defense from their attack. Only a mirror is equipped as player's weapon. UFO with aliens will move towards the player from the front, left and right. Aliens will attack the players with light ray. Players are required to reflect the light ray by the mirror, or smash the UFO by the mirror.


It teaches some properties of light, such as light travels in straight lines, the cause of shadow and reflections.


Gotcha!

Gotcha! is a leisure game with the theme of nature, and its main objective is to catch butterflies and place them on water surface.


The poster of this game explains about the rationale of surface tension.

Technical Components

Engine

Unity with C# language is used to build mobile games.

Virtual Reality (VR)

Google VR is used to develop the VR scene with Google Cardboard / VR box equipped.

Bluetooth Communication with Mi Band 3

Mi Band 3 is used as the pulse rate sensor for the prototype. Native Android Java code is used to handle the sensor and is converted and used as a Unity Android Plugin.

Game Controller

PlayStation DualShock Wireless Controller is equipped. The controller is connected to the mobile phone by Bluetooth.

Local Storage

PlayerPrefs in Unity is used to store the players' game records, so as to maintain a leaderboard.
