Introduction

In recent decades, mental health issues have become increasingly serious among students. Drift Bottle is an innovative platform that allows users to express their emotions in a message-in-a-bottle style. It enables anonymous sharing of thoughts, whether it's expressing negative emotions in a storytelling format to relieve stress or posting positive stories to inspire others and bring a smile to the community. Additionally, Drift Bottle provides support from experts who can engage in conversations with users and offer guidance.

Technical Components

Engine

Android Studio with Java language is used to build a native Android application.

Server Support

Firebase - Cloud Firestore is used to store the usersâ€™ information, such as username, password, stories shared, and messages in the conversation between the expertise and the users. Therefore, the application identifies the users by account-based.

Content Moderation & Sentiment Analysis

Recurrent Neural Network (RNN) with TensorFlow is employed for AI with two distinct models using machine learning. It tackles a text classification problem to filter malicious content contained in the stories shared by the users and to determine the mood of the stories.

Service & Multimedia

Service is used for playing the background music, and some animations are added for the splash screen and loading page.

Application Details

  • Register for or log in to an account

  • Share stories anonymously

  • Read stories of fellow peers with mood filter (choose to read happy or sad stories)

  • For pre-registered specialists:
    • Provide guidance for assigned users
  • For other users:
    • Carry out consultations with assigned specialists
