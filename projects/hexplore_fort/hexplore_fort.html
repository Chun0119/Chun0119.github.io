Introduction

Hexplore Fort is an RPG adventure game that aims to demonstrate the combination of physical and virtual gaming experiences. The equipped robot acts as the player's character in the game. Players are required to break into the mansion, fight the guardians, and capture the princess.

Technical Components

Engine

Unity with the C# language is used to build an AR Android game.

Augmented Reality (AR)

AR Core is used to provide AR-related features such as plane detection and image recognition. This technology augments the virtual terrain and characters in the game.

AR - Plane Detection

Environmental understanding performed by AR Core involves looking for a horizontal plane in reality. The virtual terrain of the game is generated and augmented onto the detected plane.

Programmable Robot

An 3D-printed Arduino-based hexapod is integrated into the project. Players can control the movement of the robot through the game, replacing the original Vorpal gamepad controller.

AR - Image Tracking

Image recognition performed by AR Core helps the game recognize the current position of the hexapod. This feature enables the game to simulate the walking of a real robot through the virtual terrain.

Bluetooth Connection

Bluetooth Classic is equipped with the Bluetooth serial board in the hexapod. It acts as a connection bridge between the game and the robot. Native Android Java code is used to handle the Bluetooth commands for controlling the robot's movement, and it is converted and used as a Unity Android Plugin.

Game Details

  1. Enable Bluetooth and connect the hexapod.

  2. Scan the environment with the camera to target a horizontal plane and generate the map.

  3. Scan the image on the body of the hexapod to recognize the player character.

  4. Explore the map with the robot:
    • Control the robot to walk or run across the terrain.
    • Pick up keys placed on the map to unlock locked doors.
  5. Fight against guardians:
    • Complete quick-time events (QTEs) to perform attack or defense actions.
  6. Upgrade the power of the character:
    • Pick up enhancing items placed on the map.
    • Make transactions with the spy using the gold earned after fighting the guardians.
  7. Reach the location where the princess is and capture her to win the game.

Future Development

Since Hexplore Fort, developed for this project, is mainly for demonstrating the combination of physical and virtual gaming experiences, the level system has been eliminated, and Hexplore Fort has only one fixed map. Taking reference from Tower of the Sorcerer, here are some suggestions for long-term development:

For example:

  • Implement a level system.
  • Add more types of enemies.
  • Introduce a skill system for both enemies and players.

As time goes on, if object detection becomes available in ARCore, the new AR technology can be used to replace the current image recognition system for the robot. This will greatly reduce the limitations of the game caused by ARCore.
